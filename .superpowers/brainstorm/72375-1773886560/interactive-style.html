公开站 Interactive 风格方向

现代 Nonprofit 网站常见的交互模式，选择你喜欢的方向

A

Scroll-Driven Storytelling

滚动驱动叙事 — 页面随滚动逐步展开内容，元素淡入、滑入、缩放。类似 charity:water、UNICEF 等大型非营利网站。

  • Hero 视差滚动 + 大标题动画
  • 数据统计区 — 数字滚动计数器（"164+ Certified Doulas"）
  • 时间线组件 — 认证步骤随滚动逐步展开
  • 卡片网格 — hover 浮起 + 阴影变化
  • Testimonial 轮播
适合：想要有"讲故事"感觉的品牌展示
参考：charity:water, Khan Academy, Teach for America
B

Component-Rich Interactive

组件式交互 — 用丰富的 UI 组件让用户参与，而非被动阅读。强调功能性交互。

  • Accordion / FAQ 折叠面板
  • Tab 切换 — 多语言内容、不同认证类型对比
  • Interactive Steps — 点击/悬停看详细说明的认证步骤图
  • 搜索 + 筛选 — 文章/培训机构可按类别筛选
  • 嵌入式证书验证 — 首页直接输入验证码查证书
适合：信息密集、用户需要快速找到答案
参考：Red Cross, ACLU, Coursera
C

A + B 结合

叙事 + 功能混合 — 首页用滚动叙事建立品牌感，内页用组件交互提供实用功能。

  • 首页：Hero 动画 + 统计计数器 + Testimonial 轮播 + CTA
  • About 页：时间线 + 团队卡片
  • 认证页：Interactive Steps + Accordion FAQ
  • 文章页：网格布局 + 分类筛选
  • 考试查询：输入编号 → 动画展示结果
适合：品牌感 + 实用性都要
效果：首页吸引人，内页留住人
