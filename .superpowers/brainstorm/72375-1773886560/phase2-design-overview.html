Phase 2 完整设计方案

公开站 + 博客 + 考试查询 — 7 个实施步骤

Step 1 — 公共布局 + 导航

共用 Header / Footer / 响应式框架

  • Header：深色导航栏（#0c2231），樱花 Logo，About Us / Programs / Become a Doula 下拉菜单 + "Get Certified" CTA 按钮
  • Footer：联系信息、社交链接、快速导航、版权
  • Mobile：汉堡菜单 + 侧滑导航
  • 字体：Poppins（标题）+ Inter（正文）
  • 品牌色 CSS 变量系统（紫色系 + Navy）

产出：公共 layout、全局样式、Header/Footer 组件

Step 2 — 首页（Scroll-Driven Storytelling）

  • Hero 区：全屏母婴背景图 + 暗色遮罩 + 标语动画淡入 + 双 CTA（"I'm a Caregiver" / "Find a Doula"）
  • Mission 区：滚动淡入，品牌介绍 + 四个核心价值 badge（Compassion, Integrity, Excellence, Knowledge）
  • 数据统计区：数字滚动计数器 — "164+ Certified Doulas" / "5+ Languages" / "6+ Insurance Partners"
  • 认证路径预览：3 步简化流程卡片，点击跳转详情
  • Testimonial 轮播：学员/客户评价
  • 联系区：紫色渐变背景 + 联系表单（还原 Framer 设计）

交互：Intersection Observer 驱动的滚动动画，无需额外动画库

Step 3 — About Us 页面组（3页）

  • /about-us：组织介绍 + 团队卡片（hover 翻转效果）
  • /about-us/history：交互式时间线（滚动展开，左右交替布局）
  • /about-us/mission-value：使命/愿景 + 核心价值观卡片动画

产出：3 个页面 + Timeline 组件 + TeamCard 组件

Step 4 — Become a Doula 页面组（6页）

左侧边栏导航 + 右侧内容（还原 Framer 布局）

  • /become-a-doula/steps-to-certification：Interactive Steps — 点击/hover 展开详情，带进度指示
  • /become-a-doula/license-and-exam：考试信息 + Accordion FAQ
  • /become-a-doula/renew-recertification：续期流程 + 在线验证入口
  • /become-a-doula/find-a-doula-training：培训机构列表（卡片式 + 筛选）
  • /become-a-doula/doula-verification：重定向到 /verify/[code]（Phase 1 已做）
  • /become-a-doula/code-of-conduct：行为准则（静态内容页）

交互：Sidebar 导航高亮当前页，Steps 组件可点击展开，Accordion 折叠面板

Step 5 — 博客系统

  • 数据库：Supabase 新建 articles 表（title, slug, content, cover_image, category, published_at, status）
  • /articles：文章列表页 — 卡片网格布局 + 分类筛选 Tab
  • /articles/[slug]：文章详情 — 富文本渲染 + 侧边目录 + 相关文章推荐
  • Admin 后台：新增文章管理页 — 创建/编辑/发布（富文本编辑器）
  • 数据迁移：从 Framer 抓取 10 篇文章内容导入 Supabase

SEO：每篇文章生成 Open Graph 元数据 + 结构化数据

Step 6 — 考试查询系统（动态）

替代 Framer 250+ 静态页面

  • 入口页：输入考试编号 + 姓名验证身份
  • 结果页：动画展示考试结果（分数、各科目成绩、是否通过）
  • 证书链接：通过的学员直接链接到证书验证页
  • 路由：/exam-home/exam-chs/[id] 保持和 Framer 一致
  • 301 重定向：旧的静态 URL 自动指向新动态页面

数据来源：Phase 1 已有的 exam_results + doulas 表

Step 7 — SEO + 部署 + 域名切换

  • 每页 meta title / description / Open Graph
  • 自动生成 sitemap.xml + robots.txt
  • 结构化数据（Organization, Article, BreadcrumbList）
  • 将 asiandoula.org 主域名指向 Next.js 站
  • Framer 旧站停用（或保留为备份）
  • 验证所有页面 + 功能正常

admin.asiandoula.org 保持不变，公开站用 asiandoula.org

总结

新页面~15 个公开页面 + 动态考试/文章页
新数据库表articles（博客）
新组件Header, Footer, Timeline, Steps, Accordion, ArticleCard, ExamQuery, RichTextEditor
交互滚动动画、计数器、轮播、折叠面板、Tab 筛选
域名asiandoula.org → Next.js 公开站
admin.asiandoula.org → 管理后台（不变）
